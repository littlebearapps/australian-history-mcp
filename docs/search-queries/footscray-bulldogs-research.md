# Footscray/Western Bulldogs Research - Search Queries

**Purpose:** Search terms for finding historical records about Footscray Football Club and Western Oval in Australian archives (Trove, PROV, State Library Victoria, etc.)

**Date Range Focus:** 1877-1997 (VFA/VFL era at Western Oval)

---

## Quick Reference - Key Dates

| Year | Event |
|------|-------|
| 1877 | Footscray Football Club founded (some sources say 1883) |
| 1879 | First use of Western Oval |
| 1886 | Joined Victorian Football Association (VFA) |
| 1898 | First VFA Premiership |
| 1899 | Second VFA Premiership |
| 1900 | Third VFA Premiership |
| 1908 | Fourth VFA Premiership |
| 1913 | Fifth VFA Premiership |
| 1919 | Sixth VFA Premiership |
| 1920 | Seventh VFA Premiership |
| 1923 | Eighth VFA Premiership |
| 1924 | Ninth VFA Premiership (left VFA for VFL) |
| 1925 | Joined VFL (with Hawthorn and North Melbourne) |
| 1954 | First VFL Premiership |
| 1989 | Renamed to "Western Bulldogs" (briefly reverted) |
| 1995 | Western Oval renamed "Whitten Oval" |
| 1996 | Permanently became "Western Bulldogs" |
| 1997 | Last VFL/AFL home game at Whitten Oval |

## Notes

- **OCR variations**: Historical newspapers may show Footscary, Fooscray (OCR errors)
- **Hyphenation**: Historical papers often used "Foot-ball", "Grand-stand"
- **VFA powerhouse**: Won 9 VFA premierships before joining VFL in 1925
- **Only VFL premiership**: 1954 (drought ended in 2016 as Western Bulldogs)
- **Ted Whitten**: "Mr Football" - Western Oval renamed Whitten Oval in 1995
