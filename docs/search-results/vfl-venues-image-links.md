### Institutions with VFL Content

- **State Library Victoria** - Largest collection via Trove

- **National Library of Australia** - Various photographic collections

## How to Access Images

1. **Trove URL**: Click to view the full record with metadata and download options
2. **Thumbnail URL**: Direct link to preview image (where available)
3. For full-resolution downloads, visit the Trove page and follow links to the source institution
4. State Library Victoria offers free downloads of many digitised items
